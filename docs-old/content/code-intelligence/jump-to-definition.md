CodePerfect can jump to the definition of almost any (valid) identifier.
Put your cursor over it, and run <cite>Navigate</cite> &gt; <cite>Go to Definition</cite>.

## Keyboard Shortcuts

| ---------------- | -------------------------------------------------------------- |
