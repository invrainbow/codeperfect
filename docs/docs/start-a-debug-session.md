After [configuring your debug profiles](/docs/setting-up-debugger), go to
`Debug` &gt; `Select Active Debug Profile...` and select a profile.

To start debugging, run `Debug` &gt; `Start Debugging`.

## Keyboard Shortcuts

| Command         | Shortcut |
| --------------- | -------- |
| Start Debugging | `F5`     |
