Once you've [configured your debug profiles](/docs/setting-up-debugger), go to
`Debug` &gt; `Select Active Debug Profile...` and select
a profile.

To start the debugger, run `Debug` &gt; `Start Debugging`.

## Keyboard Shortcuts

| Command         | Shortcut      |
| --------------- | ------------- |
| Start Debugging | <kbd>F5</kbd> |
