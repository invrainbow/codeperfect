CodePerfect can jump to the definition of almost any (valid) identifier.
Put your cursor over it, and run `Navigate` &gt; `Go to Definition`.

## Keyboard Shortcuts

| Command          | Shortcut                                            |
| ---------------- | --------------------------------------------------- |
| Go to Definition | <kbd>gd</kbd> (in Vim Normal mode) or <kbd>âŒ˜G</kbd> |
