CodePerfect can rename an identifier and all references to it. Put your
cursor over an identifier and run `Refactor` &gt;
`Rename`.

## Keyboard Shortcuts

| Command | Shortcut       |
| ------- | -------------- |
| Rename  | <kbd>F12</kbd> |
