---
title: "Go to File"
---

CodePerfect lets you fuzzy-search for a file and open it. To do this, press
`Primary+P`.

![](/go-to-file.png)

## Keyboard shortcuts

| Command                  | Shortcut    |
| ------------------------ | ----------- |
| Toggle Go To File window | `Primary+P` |
| Select file              | `Enter`     |
