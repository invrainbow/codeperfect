---
title: "Go to File"
---

CodePerfect lets you fuzzy-search for a file and open it. To do this, press
`Cmd+P`.

![](/go-to-file.png)

## Keyboard Shortcuts

| Command                  | Shortcut          |
| ------------------------ | ----------------- |
| Toggle Go To File window | `Cmd+P`           |
| Move cursor down         | `Cmd+J` or `Down` |
| Move cursor up           | `Cmd+K` or `Up`   |
| Select file              | `Enter`           |
