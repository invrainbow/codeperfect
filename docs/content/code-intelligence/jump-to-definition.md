CodePerfect can jump to the definition of almost any (valid) identifier. Just
put your cursor over it, and press <kbd>gd</kbd> or <kbd>⌘G</kbd>.

## Keyboard Shortcuts

| Description        | Key                            |
| ------------------ | ------------------------------ |
| Jump to definition | <kbd>gd</kbd> or <kbd>⌘G</kbd> |
