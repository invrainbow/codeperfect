CodePerfect can rename an identifier and all references to it. Put your cursor over an identifier and run <cite>Refactor</cite> &gt; <cite>Rename</cite>.

## Keyboard Shortcuts

| Command | Shortcut       |
| ------- | -------------- |
| Rename  | <kbd>F12</kbd> |
